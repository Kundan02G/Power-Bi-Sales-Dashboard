## 📊 Global Furniture Sales Dashboard (Power BI)

### 📌 Project Overview:
This project presents an interactive Power BI dashboard analyzing global furniture sales data. The dashboard provides insights into sales performance, profit trends, and regional distribution.

---
### 🎯 Objectives:
- Analyze total sales and profit
- Identify top-performing regions and products
- Understand sales trends over time
- Support data-driven decision making

---
### 🛠 Tools Used:
- Microsoft Power BI  
- Excel (Data Cleaning)  

---

### 📈 Key Insights:
- Identified highest sales in specific regions  
- Analyzed profit vs sales trends  
- Highlighted top-selling furniture categories  


### 📁 Files Included:
- Global Furniture Sale.pbix (Power BI file)
- Dashboard screenshots

---

### 🚀 Conclusion:
This dashboard helps in understanding business performance and can be used by stakeholders to make informed decisions.
